서버 구현 미흡으로 로그인, 계정에 따라 분리가 되어 있지 않습니다.

master에 Home 파일을 들어가시면 확인 가능합니다.

-메세지 기능
별도의 설치 및 사용방법을 올리겠습니다



-상담 예약 기능
현재는 교수님 용 계정으로 접속시 연결되는 페이지로 만들어놨습니다.

codebase : 오픈소스 코드 폴더
reser_list : 현재는 이경용 교수님만 링크를 걸어놨습니다.
성함 클릭시 plain_structure_professor (예약 기능 확인 가능)로 연결 



-교수님 정보확인 기능
현재는 학생 용 계정으로 접속시 연결되는 페이지로 만들어놨습니다.

every_list : 교수님 성함 클릭 시 S.Prof_Info로 연결
(P.Prof_Info : 교수님 계정으로 접속했을때 본인의 오피스 정보에 따라 아이콘 변경가능)



-메인 홈페이지

Home : 홈페이지 메인으로 현재 가능한 다른 기능들을 연동해놨습니다.
